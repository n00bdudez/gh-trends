# github-trends

A lightweight, static GitHub Trending page with dark/light theme toggle and daily/weekly/monthly filters.

## Features
- Dark/Light theme with local storage persistence
- Trending repos displayed with name, description, stars, and language
- Filter by Daily / Weekly / Monthly
- Repo count, language count, and last updated date
- Skeleton loaders for smooth UX
- Fully responsive

## Usage
1. Clone this repo.
2. Update `data.json` with your trending repos or automate via GitHub Actions.
3. Open `index.html` in any browser.
4. Optional: Deploy to GitHub Pages or any static hosting.
